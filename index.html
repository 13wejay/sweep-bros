How to Use

Numeric Mode (Default)

Sort files into numbered folders.

  • 0-9 Move to corresponding folder
  • Space Skip file
  • Ctrl+Z Undo last action

Arrow Mode

Classic Keep/Delete workflow.

  • ← Delete file
  • → Keep file
  • Space Skip file

View & Filter Controls

Quick access to viewing options.

  • I Toggle file info panel
  • M Toggle image filter
  • V Toggle video filter
  • A Show all file types
  • + / - Zoom in/out (images)
