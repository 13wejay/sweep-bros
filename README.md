# Sweep Bros

Sweep Bros is a fast, improved media sorting tool designed to help you organize your photos and videos into custom folders using keyboard shortcuts.

## Features

- **No Installation Required**: Works directly in your browser.
- **Privacy First**: All files stay on your computer. Nothing is uploaded.
- **Custom Folders**: Create up to 10 destination folders.
- **Quick Keys**: Use 0-9 keys to sort instantly.
- **Arrow Mode**: Classic Keep/Delete workflow with arrow keys.
- **Undo Support**: Easily undo your last action.
- **Video Support**: Auto-play videos for quick review.

## Usage

1. **Open the App**: Launch `index.html` in your web browser.
2. **Select Folder**: Click "Select Folder" and choose a directory containing media files.
3. **Sort**:
   - **Numeric Mode**: Press `0`-`9` to move the current file to the corresponding folder.
   - **Arrow Mode**: Use `Left Arrow` to delete and `Right Arrow` to keep.
   - **Space**: Skip the current file.
4. **Done**: View your statistics and start a new session if needed.
